# MusicaBot

MusicaBot は、ある特定の音楽サークルのイベント企画・参加者管理・演奏エントリー集計を劇的に効率化するために開発された Discord Bot です。
Discord 上での募集から、Google スプレッドシートへのリアルタイムな名簿作成・エントリー転記までを自動で行います。

## 🚀 主な機能

### 1. イベント募集 ＆ 参加者管理 (`/start`)
- コマンド一つで「イベント募集パネル（Embed）」をDiscordに送信します。
- 募集と同時に、指定した数の**専用テキストチャンネル**と**専用ロール**を自動生成します。
- メンバーが「参加」ボタンを押すと、入力フォーム（Modal）が立ち上がり、**「期・名字・名前」**を収集します。
- 収集したデータは、裏側で **Google スプレッドシートへリアルタイム自動転記** され、名簿が自動構築されます。

### 2. 曲の募集・管理 (`/曲募集`)
- 各ステージやバンドの募集をスラッシュコマンドからテンプレートに沿って投稿できます。
- 残り募集人数のカウントダウンや、メンバーの追加・取り消し・締め切り処理をボタン操作で直感的に行えます。

### 3. エントリー情報の自動集計
- 各チャンネルに投稿されたエントリーメッセージを自動で解析します。
- 構築された名簿（参加者一覧）と照合し、表記揺れや未登録メンバーがいないかをチェックしながら、**スプレッドシートの集計タブへ一括転記**します。
- 名簿にない名前がある場合は、Discord上で `⚠️` のリアクションや警告メッセージで通知します。

- `discord.py` (v2.x)
- `gspread` / `google-auth` (Google Sheets API 連携用)
- `Flask` (Render等のホスティング環境での常時起動ヘルスチェック用)
